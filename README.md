## Clonar o repositório
```bash
$ git clone https://github.com/FilipePaiva10/D2ControleFinanceiro.git
```

## Acessar Pasta
```bash
cd gerenciador-financeiro
```

## Instalar dependências
```bash
npm install
```

## Rodar Projeto
```bash
npm start
```
